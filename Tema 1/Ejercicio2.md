## Ejercicio 2. En la aplicación que se ha usado como ejemplo en el ejercicio anterior, ¿podría usar diferentes lenguajes? ¿Qué almacenes de datos serían los más convenientes?

La aplicación tomada como ejemplo es una aplicación web desarrollada con el framework web Django, el cual utiliza el lenguaje de programación Python. La interfaz utiliza HTML, CC y JavaScript. Por tanto la respuesta es afirmativa, puede usar diferentes lenguajes de programación.

Como persistecia utilizo la base de datos NoSQL MongoDB. En mi opinión, un almacén de datos de este tipo es el más conveniente para esta aplicación, ya que Python se integra fenomenal con ella y no necesitaos relaciones entre las entidades.
